What it includes

Cosa include

  • Zed settings for common web languages.
  • Cross-platform keybindings based on the VS Code layout.
  • Diagnostics, inlay hints, minimap, and terminal preferences.
  • Impostazioni Zed per i linguaggi web più usati.
  • Keybinding cross-platform basati sulla mappa di VS Code.
  • Preferenze per diagnostics, inlay hints, minimap e terminale.

Install

Installazione

Open Zed's Command Palette, run zed: open settings file and zed: open keymap file, then paste the downloaded config into each file.

Apri la Command Palette di Zed, esegui zed: open settings file e zed: open keymap file, poi incolla la configurazione scaricata nei rispettivi file.

Extensions

Estensioni

Recommended core: HTML, PHP, SQL, and SCSS & SASS. Snippets, LOG, Git Firefly, and the macOS Classic theme are optional extras.

Nucleo consigliato: HTML, PHP, SQL e SCSS & SASS. Snippet, LOG, Git Firefly e tema macOS Classic sono extra opzionali.
